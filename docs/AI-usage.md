# AI Usage Declaration — UniStack

## Policy
This project was developed with the assistance of AI tools (Claude by Anthropic) in accordance with the assignment's "AI as helper, not replacement" rule.

## How AI Was Used
| Area | AI Assistance | Human Work |
|------|---------------|------------|
| Boilerplate structure | AI suggested MVC folder structure | Team reviewed and adapted to project |
| CSS base styles | AI provided initial CSS variables and reset | Team customized colors, layout, and responsiveness |
| SQL schema | AI helped write CREATE TABLE syntax | Team designed table relationships and chose field types |
| Documentation templates | AI helped format markdown docs | Team wrote all real content (street report, user stories) |
| Debugging | AI helped identify syntax errors | Team wrote and tested all logic |

## What AI Did NOT Do
- AI did not conduct the street research or write the user quotes
- AI did not make architectural decisions about role separation
- AI did not write the final user stories (team derived these from real student conversations)
- AI did not test the application
- AI did not deploy the application

## Team Members and Contributions
Each team member made a minimum of 3 Git commits covering their assigned features. See Git history for full contribution breakdown.
