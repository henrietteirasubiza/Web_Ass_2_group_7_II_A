# UI Style Guide — UniStack

## Color Palette
| Name | Hex | Usage |
|------|-----|-------|
| Primary Dark | #1e3a8a | Navbar, headings |
| Primary | #1d4ed8 | Buttons, links, accents |
| Primary Light | #dbeafe | Badge backgrounds |
| Success | #16a34a | Approved badges, success alerts |
| Warning | #d97706 | Pending badges, warning |
| Danger | #dc2626 | Rejected, delete buttons |
| Gray 50 | #f9fafb | Page background |
| White | #ffffff | Card backgrounds |

## Typography
- Font: Inter (Google Fonts)
- Headings: 600–700 weight
- Body: 400 weight, 0.9–1rem
- Small/meta: 0.8–0.85rem, color #6b7280

## Components
- **Cards:** White, 10px radius, subtle shadow, hover lift effect
- **Badges:** Pill shape, color-coded by type/status
- **Buttons:** Rounded 7px, primary/success/danger/outline variants
- **Forms:** 2px border, focus highlight with primary color
- **Tables:** Striped hover, sticky header with dark background
- **Alerts:** Left-border colored alerts, auto-dismiss in 5s

## Spacing
- Base unit: 1rem (16px)
- Card padding: 1.4rem
- Grid gap: 1.25rem
- Section margin-bottom: 2rem
